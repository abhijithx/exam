# NSA Java and Shell Scripts

This repository contains Java and Shell script examples for basic programming tasks.

## Folders

- `javaCode/` - Java source files
- `shell/` - Shell script files

## How to Run Java Code
1. Navigate to the `javaCode` folder.
2. Compile: `javac LowestProductPrice.java`
3. Run: `java javaCode.LowestProductPrice`

## How to Run Shell Scripts
1. Navigate to the `shell` folder.
2. Make script executable (if needed): `chmod +x scriptname.sh`
3. Run: `./scriptname.sh`

## Notes
- `.class` files are ignored by version control (see `.gitignore`).
- Update this README with more details as you add scripts.
